# Student Result Management System

## Problem Statement
Managing student results manually is error-prone. This project automates result storage and grading.

## Features
- Add student records
- Update marks
- Auto grade calculation
- Persistent SQLite storage

## Tech Stack
- Python
- SQLite

## How to Run
python result_system.py

## Future Improvements
- Delete student option
- Subject-wise analysis
- Web-based interface
